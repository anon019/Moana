# Moana

AI-native early childhood education content generation system

## Overview

Moana is an intelligent content generation system designed for early childhood education (ages 1-3). It generates personalized picture books, nursery rhymes, and educational videos using AI.

## Features

- Personalized picture book generation
- Story narration with TTS
- Interactive learning elements
- Theme-based content (habits, cognition)
